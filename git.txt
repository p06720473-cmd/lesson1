
Основные учебные команды Git
git init — создать новый локальный репозиторий
git clone <url> — склонировать удалённый репозиторий
git status — посмотреть состояние файлов
git add <файл> — добавить один файл в индекс
git add . — добавить все изменения в индекс
git commit -m "сообщение" — сохранить коммит
git log — посмотреть историю коммитов
git diff — увидеть изменения перед коммитом
Работа с ветками
git branch — посмотреть список веток
git branch <имя> — создать новую ветку
git checkout <имя> — переключиться на ветку
git switch <имя> — переключиться на ветку (новый синтаксис)
git merge <ветка> — слить ветку в текущую
Удалённый репозиторий
git remote -v — посмотреть подключённые удалённые репозитории
git push origin <ветка> — отправить изменения на GitHub
git pull origin <ветка> — получить изменения из GitHub
git fetch — загрузить данные с сервера без слияния
Полезное для учебы
git reset --hard — отменить все несохранённые изменения (осторожно!)
git stash — временно сохранить локальные изменения
git checkout -- <файл> — откатить изменения в файле
git show <commit> — показать содержимое коммита
